How to generate results:


1. Download Extension

Go onto Chrome and download this extension.
All this does scrape the HTML from the seller's page which we will then use as our source information.

2. Scrape page

Go onto the sellers page and, if necessary, click 'continue save'.

3. Find your results

Go onto the 'Upload HTML' tab.
Either click 'upload file' and select the html file (it should be your most recently downloaded file) or just drag and drop it.
